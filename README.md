# Dostaffkin

Веб-приложение для отслеживания и управления доставками. Позволяет пользователям создавать заказы и отслеживать статус доставки в реальном времени.

## Возможности

- **Создание заказов** — оформление новых заказов доставки
- **Отслеживание доставок** — просмотр статуса и информации о текущих и прошлых заказах
- **Уведомления** — всплывающие уведомления о происходящих событиях
- **Адаптивный интерфейс** — работает на всех устройствах

## Технологический стек

- **Frontend:** Angular 21.1.1, TypeScript 5.9.3
- **Состояние:** RxJS 7.8.2
- **UI/UX:** ngx-toastr для уведомлений
- **Тестирование:** Vitest 4.0.18, jsdom 27.4.0
- **Упаковка:** npm 10.9.2
- **Форматирование кода:** Prettier

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

Приложение будет доступно по адресу `http://localhost:4200`

## Сборка

```bash
npm run build
```

Результат будет в папке `dist/`

## Разработка

```bash
npm run watch
```

Режим отслеживания изменений с автоматической пересборкой

## Тестирование

```bash
npm test
```
